Automação para emissão de notas fiscais entre estados (iguais ou diferentes), com cálculo automático de impostos baseado nas regras de tributação interestadual.
Permite cadastrar produtos, definir quantidade (litros, quilos, toneladas), valor unitário e gerar o cálculo fiscal de forma dinâmica.
Projeto não integrado ao SEFAZ, mas estruturado para futura integração.
